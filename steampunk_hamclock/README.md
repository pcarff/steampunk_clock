# Steampunk HamClock

A premium steampunk-themed clock application modeled after HamClock and the NetCarff Steampunk Clock Android app.

## Features
- **Nixie Tube Display**: Real digit images from the original Steampunk Clock app.
- **Smooth Transitions**: Animated digit changes using Framer Motion.
- **FastAPI Backend**: Robust Python backend for telemetry and future GPS/Solar integration.
- **React Frontend**: Modern, high-performance UI.

## Getting Started

1. Run the start script:
   ```bash
   ./start.sh
   ```

2. Access the application:
   - Frontend: `http://localhost:5173`
   - Backend API: `http://localhost:8000`

## Structure
- `backend/`: FastAPI application.
- `frontend/`: React + Vite application.
- `assets/`: Design assets and textures.
